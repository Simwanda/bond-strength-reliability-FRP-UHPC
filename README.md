# Bond Strength Reliability Calibration for Steel and FRP Reinforcement

This repository contains supplementary material accompanying the journal paper on simplified bond strength equations and reliability-based partial factor calibration.

## Contents

- Curated experimental bond database
- Simplified regression equations per subgroup
- Cross-validation results
- Lognormal model uncertainty parameters
- Calibrated partial factors (Î² = 3.0, 3.5, 3.8)
- Publication figures

Python version: 3.10+
Required libraries:
- numpy
- pandas
- scipy
- matplotlib
- scikit-learn
